# MoodBoard Me

一个基于 Next.js 开发的心情板生成器，帮助用户可视化他们的情绪状态。

## 功能特点

- 💭 心情输入：用户可以描述当前的心情
- 🖼️ 图片生成：根据心情生成匹配的图片
- 💬 名言推荐：提供励志名言
- 🎵 音乐推荐：推荐符合心情的音乐
- 🌓 深色模式：支持明暗主题切换
- 📱 响应式设计：完美适配各种设备

## 技术栈

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Radix UI
- Lucide Icons

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
npm run start
```

## 部署

本项目已配置为可以直接部署到 Vercel 平台。
